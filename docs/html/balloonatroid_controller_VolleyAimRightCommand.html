execute

function execute (
note:INotification) : Void

Moves the Volley Balloon to the Right.

Retrives the VolleyProxy proxy to determine the
current state of the Volley Balloon. If the in the DORMANT state
then it is moved by AIM_AMOUNT pixels specified in the
VolleyDO data object if the balloon will end up within bounds.
If in the INFLIGHT state a horizontal speed of STEER_AMOUNT
pixels per second is added to its speed in the aiming direction.

Movement is accomplished by setting the targetX and target properties of the
vollery ballon. The current valley state update method iteratively moves the balloon
to the target based on a movement algorithm.
See also:
  • balloonatroid.model.VolleyProxy
  • balloonatroid.model.volley.VolleyDO.AIM_AMOUNT
  • balloonatroid.model.volley.VolleyDO.STEER_AMOUNT
Usage:
  • VolleyProxy and GameProxy need to be registered
    prior to command execution.
