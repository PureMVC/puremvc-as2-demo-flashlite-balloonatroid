PlayScreenMediator

Kind of class:class
Inherits from:Mediator
Implements:
  • IMediator
Classpath:balloonatroid.view.PlayScreenMediator
File last modified:Monday, 17 March 2008, 14:34:46
A Mediator for interacting with the Splash Screen.

Mediates the UI for the Play Screen by adding buttons, initiating entry and exit
animations to the screen, registering and unregistering button press, keydown,
and softkey command handlers. The level MovieClip is loaded on the first frame of the entry animation.
Acts as the view screen counterpart for the PLAY state.

Instance methods

handleNotification

function handleNotification (
note:INotification) : Void

Handle all notifications this Mediator is interested in.

Called by the framework when a notification is sent that
this mediator expressed an interest in when registered.

  • Plays play screen entry animation on receipt of ENTER_PLAY.
    The level MovieClip is loaded on the first frame of the animation.
  • Registers UI buttons, boss and volley balloons on receipt of PLAY_ENTER_DONE.
  • Actions on PLAY_EXIT_DONE not implemented yet.
  • Unregisters and removes UI buttons on receipt of EXIT_PLAY.
Parameters:
INotification:
a notification
See also:
  • listNotificationInterests
  • in Balloonatroid.fla
