BossSpriteMediator

Kind of class:class
Inherits from:Mediator
Implements:
  • IMediator
Classpath:balloonatroid.view.BossSpriteMediator
File last modified:Monday, 17 March 2008, 16:04:11
A Mediator for interacting with the Boss Sprite.
Adds itself to the sprite entities collection on onRegister.
Registers EnterFrame handler and updates its current state.
Listens to BIRTH_DEFENDER notifications and births Defender Balloons.

handleNotification

function handleNotification (
note:INotification) : Void

Handle all notifications this Mediator is interested in.

Called by the framework when a notification is sent that this mediator expressed an interest in when registered.
Births Defender Balloon on receipt of BIRTH_DEFENDER notification.
Defender is created by duplicating the master defender instance in LevelXX.fla (level MovieClip).
Initial position of defender is the registration point of the Boss.
DefenderSpriteMediator and DefenderProxy
names are set to instance name of the defender MovieClip instance.
Defender immediately goes into the ATTACK state.

Parameters:
INotification:
a notification
