File last modified:Monday, 17 March 2008, 10:46:14
A Mediator to help with debugging Notifications.

Add all notification interests to the aNotificationList Array.
Intercepts all interested notifications and:
  1. Appends them to the 'txtStatus' text field instance on the _root timeline (if it exists).
  2. Traces them to the output panel.
  3. Logs them to Xray.
Handles special notifications.
  • balloonatroid.GameFacade.DEBUG_ERROR Notification body is logged as an ERROR on Xray.
  • balloonatroid.GameFacade.DEBUG_INFO Notification body is logged as INFO on Xray.
  • balloonatroid.GameFacade.CLEAR_STATUS Notification body erases the 'txtStatus' field.
