File last modified:Thursday, 20 March 2008, 11:44:10
Proxy for interacting with the Boss Balloon Data Object.
Inherits the IStateMachine interface implementation to access the boss state machine.
Currently implements the boss:
  • Wander state (back and forth movement).
To do:
  • Implement the 'roam', 'attack', 'evade', and 'hit' states.
