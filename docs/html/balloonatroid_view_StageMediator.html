A Mediator for interacting with the stage.

Top level Mediator for the game. Registers the following child mediators for each game screen.

  • SplashScreenMediator
  • PlayScreenMediator
  • DebugHelperMediator

onRegister

function onRegister (
) : Void

Called by the View when the Mediator is registered.

Registers the game screen mediators. DebugHelperMediator is
registered only if balloonatroid.GameFacade.debug is set to true.
Assigns a local reference to balloonatroid.model.GameProxy to gameProxy.
Registers enterFrameHandler method as an EnterFrame event handler to the stage.
Usage:
  • Expects balloonatroid.model.GameProxy to be registered already.
