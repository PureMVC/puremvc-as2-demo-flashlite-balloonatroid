execute

function execute (
note:INotification) : Void

Add sprite to sprite entities collection for hit testing.

Usually called from sprite mediator during initialization. Retrieves the
SpriteEntitiesProxy and adds the sprite passed in the
note body to the sprite entities collection. The sprite should
subclass AbstractSpriteDO.
Parameters:
note:
Notification with type AbstractSpriteDO object as body
See also:
  • balloonatroid.model.SpriteEntitiesProxy
  • balloonatroid.model.AbstractSpriteDO
Usage:
  • SpriteEntitiesProxy needs to be registered prior to command execution
