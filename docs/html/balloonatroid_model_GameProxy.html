Classpath:balloonatroid.model.GameProxy
File last modified:Thursday, 20 March 2008, 11:49:18
Proxy for interacting with the Game Data Object.
Implements the IStateMachine interface to access the game state machine.

Currently implements the game:

  • Splash screen state.
  • Play screen state.
To do:
  • Implement the 'choose level', 'help/info screen', and 'buy more levels' states.
