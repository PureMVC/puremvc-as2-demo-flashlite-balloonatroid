VolleyProxy

Kind of class:class
Inherits from:AbstractSpriteProxy < patterns.proxy.Proxy
Implements:

  • IStateMachine
Classpath:balloonatroid.model.VolleyProxy
File last modified:Thursday, 20 March 2008, 12:06:29
Proxy for interacting with the Volley Balloon Data Object.
Inherits the IStateMachine interface implementation to access the volley state machine.

Currently implements the volley:

  • Dormant state (recovered and waiting to volley).
  • In flight state (volleyed and moving up).
  • Recover state (went off screen and in recovery mode).
